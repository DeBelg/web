## Development

### Install

Requirements:
- [Node.js](https://nodejs.org/en)
- or [Bun](https://bun.sh/)

Clone repository and install:

```
git clone git@github.com:web3privacy/w3pn-website.git
npm install
```

### Run local dev server

```
npm run dev
```

### Build production

```
npm run build
```
